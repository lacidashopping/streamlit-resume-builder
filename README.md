# 📄 Dynamic Resume Builder with 2x2 Photo Support

A lightweight, interactive web application built with **Streamlit** and **python-docx** that allows users to create elegant, professional, and editable Microsoft Word (`.docx`) resumes in real-time. 

Live App Link: [ramer-resumebuilder.streamlit.app](https://ramer-resumebuilder.streamlit.app/)

---

## ✨ Features

* **Dynamic Experience Blocks:** Add or remove multiple work experience dynamically using state management.
* **2x2 ID Passport Photo Layout:** Upload a photo to instantly format a clean, borderless side-by-side header grid.
* **Real-time Word Parsing:** Automatically formats standard plain text description boxes into clean Word document bullet points.
* **In-Memory Buffer Processing:** Generates and prepares files entirely in-memory using binary bytes streams—meaning fast downloads with zero temporary server file clutter.
